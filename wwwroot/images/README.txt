PROFILE PHOTO - Required for your image to show on the Home page

1. Put your photo in THIS folder (copy-paste or save here):
   C:\Users\HP\Desktop\Portfolio\wwwroot\images\

2. Rename the file to exactly:  profile.jpg
   (All lowercase. If you use .png, name it profile.png and in Admin > Edit Resume
    set "ProfileImageUrl": "/images/profile.png" in the Contact section.)

3. In File Explorer, confirm you see:
   C:\Users\HP\Desktop\Portfolio\wwwroot\images\profile.jpg
   (File size should be > 0 KB.)

4. Save, then refresh the portfolio in the browser (Ctrl+F5).

If you still see "AV" instead of your photo: the file is missing or in a different
folder. This folder must contain a file named profile.jpg (or the URL you set).
Recommended: square image, at least 400x400 pixels.
